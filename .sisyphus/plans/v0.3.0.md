- **Librarian (pi-agent-core)**: PendingMessageQueue with "all"|"one-at-a-time" modes, 3-phase parallel execution (prepareToolCall→executePreparedToolCall→finalizeExecutedToolCall), system prompt is string|function union (no Mustache), steering polled mid-loop, follow-up polled at natural stop

### Metis Review
**Identified Gaps** (addressed):
- Retry middleware: `apply_on_error` needs `conversation` parameter — resolved: pass current conversation
- UX-3 cancellation: pi uses `Promise.all` (no early exit on single failure) — resolved: adopt same (all tools run to completion)
- UX-5d escape hatch: raw record constructor must be made private — resolved: make_agent is only path
- UX-4 Mustache: no lambdas/partials (arbitrary code exec risk) — resolved: variable substitution only
- Tool-1f edit overlap: algorithm unspecified — resolved: left-most-wins, reject on conflict
- Metrics overflow: 32-bit counter risk — resolved: use 63-bit OCaml int (no external dep)
- Binary file handling for read tool — resolved: detect + base64 encode
- grep regex timeout on large files — resolved: Cancellation.with_timeout 30s, partial results on timeout
